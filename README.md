# EmergencyAid

EmergencyAid is an Android mobile application developed as a capstone project. It helps users respond to emergency situations by providing AI-assisted guidance, first aid learning modules, emergency contacts, and record management.

## Features

-  User Authentication using Firebase Authentication
-  AI-powered Emergency Chatbot
-  First Aid Learning Modules
-  Quiz Assessment
-  Emergency Contact Directory
-  Emergency Record History
-  User Profile Management
-  Admin Dashboard

##  Technologies Used

### Programming Languages
- Kotlin
- PHP

### Frameworks & Libraries
- Android SDK

### AI & APIs
